## §A.1 작성 규약

- **너비**: desktop 80자, tablet 60자, mobile 360px (35자 wrap)
- **문자**: box-drawing characters만 사용 (`┌ ─ ┐ │ └ ┘ ├ ┤ ┬ ┴ ┼ ═ ║`)
- **breakpoint**: desktop ≥1280 / tablet 768~1279 / mobile 360~767
- **색상 표기**: 텍스트에 `[GREEN]` `[YELLOW]` `[ORANGE]` `[RED]` 마커로 명시 (실 구현 시 Tailwind variant)
- **데이터 입력**: 각 모형 상단에 "in: …" 형식으로 audit-core 함수/필드 명시
